<h1> RodOfRoll </h1>
<p>Es un proyecto Final de FP Superior de DAM. Que consiste en una aplicación para gestionar partidas de Rol.</p>
<h3>Tecnologías</h3>
<ul>
  <li>AndroidStudio</li>
  <li>Firebase</li>
</ul>

  </ul>
